# Ai-assistant
### A Simple Python Speech-To-Work program. This program will recognise and analyze your voice and perform the specific tasks.

#### Tasks it can perform:
- Search Wikipedia
- Play something
- Open Youtube
- Open Google
- Open Gmail
- Open Linkedin
- Open Github
- Open Spotify
- Open Instagram
- Take Screenshot
- Remeber events
- Cpu and Battery update
- Current Date
- Current Time
